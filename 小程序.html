          • 安装request-promise
            • 在云函数项目下创建一个云函数

            • 安装npm install --save request-promise 安装request-promise

            • 在创建好的云函数index.js文件中引入 var rp = require('request-promise')
            • 使用函数 rp('http://www.google.com').then().catch()
      • 协议支持
        • 根据第三方库支持
      • 是否备案
        • 可以不备案

        • 无限制
  • 组件
    • 视图容器
      • scroll-view #可滚动视图区域.使用竖向滚动时,需要给scroll-view一个固定高度,通过WXSS设置height.
        • bindscrolltoupper #滚动到顶部/左边时触发
        • bindscrolltolower #滚动到底部/右边时触发
        • 在滚动scroll-view时会阻止页面回弹,所以在scroll-view中滚动,是无法触发onPullDownRefresh
        • 若要使用下拉刷新,请使用页面滚动,而不是scroll-view,这样也能通过点击顶部状态栏回到页面
    • 开发能力

    • 表单组件
      • input #输入框
        • value
        • type
        • placeholder
        • bindinput #键盘输入时触发,event.detail = {value,cursor,keyCode}
        • bindfocus #输入框聚焦是触发,event.detail = {value:value}
        • bindblur #输入框失去聚焦时触发,event.detail = {value:value}

        • 在input聚焦期间,避免使用css动画
      • button

        • open-type #getUserInfo_获取用户信息_可以从bindgetuserinfo回调中获取到用户信息
