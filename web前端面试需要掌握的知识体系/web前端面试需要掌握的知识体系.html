      • 1.js规定了几种语言类型

      • 2.js对象的底层数据结构是什么
        • 基本数据类型存储在栈(stack)中
        • 引用类型存储在堆(heap)中
        • 栈是自动分配相对固定大小的内存空间,并由系统自动释放
        • 堆内存是动态分配,内存大小不一,也不会自动释放
        • 栈是先进先出,队列是先进后出
        • 数据在内存中的物理存储结构为:顺序存储结构和链式存储结构
        • 顺序存储结构:是把数据元素存放在地址连续的存储单元里，其数据间的逻辑关系和物理关系是一致的。数组就是顺序存储结构的典型代表。
        • 链式存储结构:是把数据元素存放在内存中的任意存储单元里，也就是可以把数据存放在内存的各个位置。这些数据在内存中的地址可以是连续的，也可以是不连续的。链表就是顺序存储结构的典型代表。
      • 3.Symbol类型在实际开发中的应用,可手动实现一个简单的Symbol
      • 4.js中的变量在内存中的具体存储形式
        • 基本数据类型保存在栈内存中,按值访问
        • 引用数据类型保存在堆内存中,指针存在栈中,根据栈内地址找到堆内存的位置
      • 5.基本类型对应的内置对象,以及他们之间的装箱拆箱操作
        • String(),Number(),Boolean()
        • Object(),Function(),Array(),Date(),Error(),RegExp()
        • Symbol必须通过Symbol()获取,因为灭有拆箱和装箱
        • 隐式类型转换(装箱)
          • 读取基本类型的值,后台创建一个临时的该基本类型所对应的包装类对象
          • 在该对象上调用方法
          • 执行方法后该对象立马销毁
          • var item = 123;item.toString();等级于var t= new Number();t.toString();t = null;
        • 显示类型转换(装箱)
          • 用内置对象如Number,Boolean,String,Object等对基本类型进行显示装箱
          • var bool = new Boolean()\
        • 拆箱
          • 把引用类型转换为基本类型
          • 拆箱内部过程是调用一个抽象方法toPrimitive(val,type),该方法第一个参数接收转换的参数,第二个参数非必须,默认number类型,为对象呗期待转换的类型
          • 本质
            • 1.如果目标对象是object,则会先执行valueOf()方法,如果结果为原始值(基本数据类型),则返回此结果
            • 2.否则再调用toString方法,如果得出为原始值则返回结果,否则抛出异常
      • 6.理解值类型和引用类型
      • 7.null和undefined的区别
        • Null类型，代码空值，代表一个空对象指针，使用typeof运算得到object
        • undefined:Undefined类型
      • 8.至少说出三种判js数据类型的方式,优缺点,如何准确的判断数组类型
        • 1.typeof：typeof xx/typeof(xx) 可以判断基本类型，null返回object，function函数返回‘function’，引用类型返回object
        • 2.instanceof:obj instanceof Object 判断引用类型，返回布尔值判断，检测Object.prototype是否存在于参数obj的原型链上
        • constructor:arr.constructor === Array 指向构造函数
        • Object.prototype.toString.call() #[Object object]
        • Array.isArray(val)
      • 9.可能发生隐士类型转换的场景以及转换原则,应如何避免或巧妙的应用
      • 10.出现小数精度丢失的原因,js可以存储的最大数字,最大安全数字,js处理大数字的方法,避免精度丢失的方法
        • 精度丢失原因:JS遵循IEEE754规范,采用双精度存储,占用64bit,也就是8位.二进制存储十进制小数时不能完整表示小数.超出最大整数也会丢失
        • 避免精度丢失:
          • 计算小数时,先乘以100或1000,变成整数再运算
          • 如果超出了安全整数,使用最新提案,BingInt大整数
